# 常見問題

{% hint style="info" %}
**BNB48 總部在哪裡？**

BNB48 完全以去中心化的方式運作，沒有總部。
{% endhint %}

{% hint style="info" %}
**BNB48 是大戶專屬俱樂部嗎？**

雖然BNB48有很多大戶會員，但不是只有大戶才能加入。
{% endhint %}

{% hint style="info" %}
**BNB48 是幣安的嗎？**

不是。
{% endhint %}

{% hint style="info" %}
**BNB48有幣安的投資嗎？**

幣安在很多方面給過BNB48一定的支持，但沒有資金支持。BNB48的資金全部自籌。
{% endhint %}
